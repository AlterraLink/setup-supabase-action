# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-07

### Added
- Initial release
- Support for multiple package managers (npm, yarn, pnpm, bun)
- Node.js setup with configurable version
- Intelligent caching for all package managers
- Supabase CLI installation
- Optional dependency installation
- Working directory support

### Features
- Automatic cache detection for each package manager
- Configurable package manager versions
- Frozen lockfile installs for reproducibility
